# Mochi-Dashboard
Dashboard for Mochi discord bot

## Installation

### Setup the virtual environement

- On Windows: \
    ```> python -m venv .env``` \
    ```> .env/Scripts/activate``` \

### Install all dependencies

- On Windows: \
    ```(.env) > python -m pip install --upgrade pip``` \
    ```(.env) > python -r pip install -r requirements.txt```
